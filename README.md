# GB Price Calculator

This price calculator for Gracie Bradenton helps us quickly figure out a customer's "out-the-door" price. For each program we offer two monthly tiers (Standard and Premium), a flat Family Plan rate for 3+ people, and prepaid 6-month and 12-month options.

- Jiu Jitsu Premium: 220/mo with no enrollment fee
- Jiu Jitsu Family Plan: 400/mo (3+ people, no enrollment fee)
- Jiu Jitsu 6-month up front 997
- Jiu Jitsu 12-month up front 1797
